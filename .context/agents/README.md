# Agent Playbooks

This directory contains playbooks for AI agents working on the Task Management System. Each playbook provides context, responsibilities, and guidelines for a specific type of development task.

## Available Agents

| Agent | Description | Primary Focus |
|-------|-------------|---------------|
| [Architect Specialist](./architect-specialist.md) | System design and patterns | Architecture decisions |
| [Backend Specialist](./backend-specialist.md) | Server-side development | APIs, database, auth |
| [Bug Fixer](./bug-fixer.md) | Defect resolution | Root cause analysis |
| [Code Reviewer](./code-reviewer.md) | Code quality review | Standards compliance |
| [Database Specialist](./database-specialist.md) | Database management | Schema, queries, migrations |
| [DevOps Specialist](./devops-specialist.md) | Deployment and infrastructure | CI/CD, environments |
| [Documentation Writer](./documentation-writer.md) | Technical writing | Docs, READMEs, guides |
| [Feature Developer](./feature-developer.md) | End-to-end feature development | New functionality |
| [Frontend Specialist](./frontend-specialist.md) | UI development | React, components, state |
| [Mobile Specialist](./mobile-specialist.md) | Mobile optimization | Responsive, touch |
| [Performance Optimizer](./performance-optimizer.md) | Performance tuning | Speed, efficiency |
| [Refactoring Specialist](./refactoring-specialist.md) | Code improvement | Technical debt |
| [Security Auditor](./security-auditor.md) | Security review | Vulnerabilities, compliance |
| [Test Writer](./test-writer.md) | Test development | Unit, integration tests |

## How to Use These Playbooks

1. **Identify the task type** - Determine which agent best matches your task
2. **Read the playbook** - Review the agent's mission, responsibilities, and best practices
3. **Follow the checklist** - Use the collaboration checklist to ensure completeness
4. **Reference key resources** - Use the linked files and documentation
5. **Document outcomes** - Fill in hand-off notes when work is complete
